# Selection Menu

various options according to input value

## Repeating structure

for; while; do while

* - [1] - Imprimir os números ímpares de 0 até 100 - FOR
* - [2] - Encontrar o nome secreto contendo 5 tentativas - DO-WHILE
* - [3] - Imprimir todos os números que conten o algoratimo '1' de zero até o limite dado pelo usuario e contar quantos '1' do intervalo - WHILE
